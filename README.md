# Sweet Delights 🍰

A modern, responsive landing page for a fictional cake & pastry shop, **Sweet Delights** — _"Freshly baked happiness delivered to your door."_

Built with **React + Vite**.

## Design

- **Brand color:** `#F6BCD1` (soft pink)
- **Fonts:** [Plus Jakarta Sans](https://fonts.google.com/specimen/Plus+Jakarta+Sans) (headings) & [DM Sans](https://fonts.google.com/specimen/DM+Sans) (body)

## Sections

- Sticky header with cart counter
- Hero with call-to-action
- Our Signature Flavours (product cards)
- Delightful Pastries gallery
- Behind the Scenes
- Freshness Delivered Quickly banner
- Customer testimonials
- Footer

## Getting started

```bash
# install dependencies
npm install

# start the dev server
npm run dev

# build for production
npm run build

# preview the production build
npm run preview
```

Then open the URL printed in the terminal (default: http://localhost:5173).

## Tech stack

- [React 18](https://react.dev/)
- [Vite 5](https://vitejs.dev/)
- Plain CSS (custom properties / variables)
